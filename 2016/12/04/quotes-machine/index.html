quotes machine

哎，这次做free code camp的编程作业，简直太苦逼了，没有想到会出现这么
多问题。

编程要求说明

  1. 一个按钮负责转换引语，按下之后它会给出另外一个随机引语
  2. 另外一个按钮可以发送一个推文向其他人分享

任务流程

页面文档

<body id="body">
<h1>my quotes</h1>
<hr>
<div>
<div id="quotes">

</div>
<button onclick="another()" id="another">another</button>
<button onclick="tweet()" id="tweet">tweet</button>

</div>

转换按钮

  1. 创建一个引语列表，比如是一个含有十个引语字符串的列表quote=[....]；
  2. 随机函数var index = Math.floor(Math.random() * (quote.length));
  3. 向页面添加引语 document.getElementById("quotes").innerHTML = quote[index];
window.onload=function(){

var index = Math.floor(Math.random() * (quote.length));
document.getElementById("quotes").innerHTML = quote[index];

};

function another() {
var two = Math.floor(Math.random() * (quote.length));
document.getElementById("quotes").innerHTML = quote[two];



}

在这里我想在页面加载之后就会有引语出现，而不是必须按下按钮之后才会出现，所以我用了window.onload= function(){}

推特按钮

说实在的，我在这个按钮上面几乎花费了4天功夫，可能会有20多个小时吧

  1. 首先是一个可以发送推文的链接，之前我想的是任意一人的推特账户，所以需要tweete api，
    再这磨蹭了一天毫无所获之后，我开始在gitter开始请教这个问题，有人推荐了tweete button，

    <a class="twitter-share-button" href="https://twitter.com/intent/tweet?text=……"> Tweet</a>

  2. 现在知道怎能推送了，然后需要解决的问题是如何在推文中自动添加此刻出现的引语，
    我在想可不可以直接在链接中增加，比如
    失败

    那我写一个点击方法把。

    $("#button").attr("href,"http://")
    哦，它是jquery还是不行，那我把改成js把，setArribrute
