svg

发表于 2017-02-27  |  2017-02-27   |   阅读次数

SVG为可缩放矢量图形，使用xml格式定义图像。

优点

  • 可压缩性高，图片体积小
  • 图片质量高，不失真
  • 可互动
  • 浏览器渲染

基本知识

元素 xml描述
line 线条
polyline 多个线构成的形状
reat 矩形
circle 圆形
ellopse 椭圆
polygon 多边形
path 任意路径

  1. 创建直线

  2. 创建矩形

  3. 创建圆形

  4. 创建椭圆

  5. 利用path创建三角形

在最后的Z代表封闭图形，若无，则呈现出两条线，无法构成三角形

stroke-dasharray 定义虚线
