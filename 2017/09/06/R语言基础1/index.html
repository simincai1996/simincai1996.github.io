向量

向量值可以是数字、字符串、逻辑值，但是必须是同一种类型。

c(1,2,3)
c("a","b","c")
c(TRUE,FALSE)

赋值

x <- c(1,2,3)
y <- c("a","b","c")
z <- c(TRUE,FALSE)

内置函数

sum(1,2,3)
help(sum)
example(sum)
list.files() //列表显示当前目录下的文件
source(test.R) //运行R文件

向量取值

p<-c("one","two","three","four")
p[1]
p[1,4]
p[1:4]

R语言与其他编程语言不同的一点是：索引从1开始

向量命名

q <- c(1:3)
names(q)<- c("first","second","third")
q
q["first"]
q["first"] <- 4
q["first"]

运算

a <- c(1,2,3)
a + 2
a - 1
a*2
a/2
b <- c(4,5,6)
a + b
a - b
a == b //比较a、b各个值的大小，结果用TRUE、FALSE 表示
a < b

m <- c(1,2,3)
n <- sin(m)
plot(m,n) //横轴、竖轴
