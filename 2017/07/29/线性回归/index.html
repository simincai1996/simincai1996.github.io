机器学习中的线性回归

发表于 2017-07-29  |  2017-07-29   |   阅读次数

自己对机器学习很感兴趣,于是在cousera上选了一门机器学习的入门课程:华盛顿大学的”机器学习案例研究”.它从机器学习的
基本使用案例出发,然后对机器学习的方法加以说明,最后可以便过编程进行练习.当然这门课程也有一个好处是它有中文字幕,方便入门
学习.

机器学习的定义

机器学习很热门,可是机器学习究竟是什么?它有没有一个点定义呢?

A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E.

在课程中,提出了这样的概念: 数据 -> 机器学习方法 -> 智能.

机器学习是人工智能的一个分支，是实现人工智能的一个途径，即以机器学习为手段解决人工智能中的问题。

机器学习的分类

在一般讲解中,通常会把机器学习分为监督学习,无监督学习.另外还有半监督学习和增强学习.

  • 监督学习从给定的训练数据集中学习出一个函数，当新的数据到来时，可以根据这个函数预测结果。
    监督学习的训练集要求是包括输入和输出，也可以说是特征和目标。训练集中的目标是由人标注的。常见的监督学习算法包括回归分析和统计分类。
    举个例子,预测房价,我通过它的面积和它的地理位置,房间个数这些特征来预测房价,我可以得到房地产一些真实的数据,因此叫做监督学习.

  • 无监督学习与监督学习相比，训练集没有人为标注的结果。常见的无监督学习算法有聚类。

    数据

    在机器学习中,通常会把数据进行分组,分为训练集,测试集.我们可以根据测试集的准确度对训练集所得的
    模型不断优化,从而得到一个准确而又适用范围广的模型.

    线性回归

    f(x) = w0 +w1* x

    在线性回归中,我们可以根据数据作出无处条线作为我们的模型,可是哪一条线是最好的呢?这就需要一个手段:
    残差分析.利用残差分析,我们找到拟合程度最好的一条直线作为我们的模型.但是对于复杂的模型来说,我们可能需要进行十二项式拟合才能得到残差最小的结果.
    但是如果这样,这个模型的适用范围可能就会变得很小,所以我们通常会拒绝这种太过复杂的模型,我们需要减少这种
    过度拟合的现象.与之相对的还有欠拟合.

那么如何避免过度拟合呢?提出了这样一种方法,将数据分为训练集,测试集.我们需要减小测试集的测试误差,获得一个准确度和广泛程度都比较高的模型.

另外我们需要注意,并不是一元线性回归拟合出一条直线才是线性回归,多项式拟合,高斯拟合这些非线性以及多元特征拟合的平面或者空间都可以算作是线性回归.

总结

  1. 了解通过输入特征来预测输出的回归模型
  2. 将数据分割为训练集,测试集
  3. 通过残差最小化预估模型参数
  4. 通过测试误差来避免过度拟合
  5. 使用多元特征来描述一个回归模型
  6. 了解回归模型的适用性
